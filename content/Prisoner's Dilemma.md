# Definition:
[[Game Theory]]: Two player **non-[[Zero Sum Game|zero sum]]** finite deterministic perfect information game!

Two prisoners can either choose to snitch (defect) on the other prisoner or cooperate (stay silent).
- If both cooperate, they get light punishment.
- If one defects, they get to leave while the other serves.
- If both defect, they both get reduced sentences.

| A / B     | Cooperate | Defect  |
| --------- | --------- | ------- |
| Cooperate | -1 / -1   | -9 / 0  |
| Defect    | 0 / -9    | -6 / -6 |

---
# Notes:

- The best scenario in aggregate is if both stay silent.
- Defecting is always a **dominating strategy** (always better regardless)
- The game theory optimal solution is to actually defect for both players. [[Nash Equilibrium]] is thus (Defect, Defect) which is the worst aggregate result.

- Note to break out of this, we can change the game payoffs
	- [[Utility]] rewards can incorporate the other person's prison time
	- We can create mechanisms to punish bad behaviours (Snitches get Stitches)
