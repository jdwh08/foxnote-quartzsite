    Linear Regression

    Linear Regression

    Sep 17, 20243 min read

    • math/linear_algebra
    • ds/ml/linreg

    Definition:

    A type of Regression, which estimate y from x1​...xn​ using β0​...βn​ with form y=β0​+β1​x1​+...+βn​xn​+ϵ. (i.e., linear)

    • where β1..n​ are the regression coefficients Weights
    • where β0​ is the intercept (Bias)

    Notes:

    Closed form solutions:
    • βi​=Var(Xi​)Cov(xi​,yi​)​ (Covariance divided by Variance)
    • (or in matrix form: β=(XTX)−1XTY)

    Feature Engineering:

    Note that our features x1​... can be things like polynomials (x2) because it’s linear with respect to our βs.

    Assumptions for Linear Regression:
    1. Linear DGP: True Data Generating Process is of the form y=β0​+β1​x1​+...βn​xn​+ϵ. (LINEARITY)
    2. No Perfect Multicollinearity: No feature xi​ can be written as the linear combination of other features xj​ and xk​ (AKA, the design matrix X must have full Matrix Rank.)
      1. Otherwise, we lack identification of the effect of xi​ as opposed to xj,k​.
    3. (weak) Exogenous Features: Features X are not correlated with errors ϵ.
    4. (strict) Exogenous Features: Errors ϵ have conditional mean of zero: E[ϵ∣X]=0.
      1. This implies that errors also have mean zero: E[ϵ]=0. (Law of Total Probability)
      2. This implies that Cov(ϵ,X)=0 since Covariance definition as E[XY]−E[X]E[Y]
    5. Homoskedasticity: Errors ϵ are constant across predictor values X.
      1. Technically, we don’t need this assumption to do the operations and have the estimators β^​ be unbiased; however, the errors Var(β^​) will be underestimates.
    6. Independence of Errors / No Autocorrelation: Errors ϵ are independent from one another between i=1,...,n, so E[ϵi​ϵj​∣X]=0 ∀ i=j
      1. We can also write the above two criteria as Var(ϵ∣X)=σ2In​
    7. (Optional): Standard Normal Errors: ϵ∣X∼N(0,σ2In​).
      1. Helpful for getting equivalence with Maximum Likelihood Estimation β^​MLE​.
      2. This does NOT mean that Y∼N! For example, the X’s could be different…

    We can achieve #6 with iid observations of features and target variables.

    Finite Sample Properties

    Unbiased if strictly exogenous: E[β^​∣X]=β, E[s2∣X]=σ2 (TODO: Proof)

    Linear Regression with all the properties above is the BLUE (BEST UNBIASED LINEAR ESTIMATOR) according to the Gauss-Markov Theorem, i.e., for all possible estimates of β^​ which are unbiased, the ones from β^​OLS​ have the lowest Variance.

    From the Maximum Likelihood:

    Recall that based on Uncertainty (Supervised ML), we see that for regression maximizing the log likelihood will minimize the squared error.
