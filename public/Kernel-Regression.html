Kernel Regression

Mar 19, 20241 min read

  • stats
  • ds/ml/linreg

Definition:

A type of Nonparametric Statistics based method which uses the training data to predict continuous values in Regression.

We weight values based on how far away they are compared to the actual value we are predicting. This depends on your choice of Kernel.
