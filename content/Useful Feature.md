# Definition:

$x_i$ is a **Useful Feature** if it helps reduce [[Loss Function]] **for a given ML model**

---
# Notes:
- Differs from the [[Relevant Feature]], except in the particular case where the given ML model is the [[Bayes Optimal Classifier]].

---
# Examples:
A constant feature allows us to represent the intercept in [[Linear Regression]] or a [[Perceptron]], and is thus **useful** without being **relevant**.
