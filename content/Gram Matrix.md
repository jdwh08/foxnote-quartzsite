# Definition:
Given some vectors $v_1 ... v_n$, a Gram Matrix is a [[Hermitian Matrix]] where each value is the [[Dot Product]] between the vectors.

Used to determine non-[[Linear Dependence]].
- Independent iff [[Matrix Determinant]] must be nonzero.
