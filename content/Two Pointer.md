# Definition:
An [[Data Structures and Algorithms|algorithm]] to iterate through an [[Array]], [[ArrayList]], or [[Linked List]].

Typically this allows for solving in [[Big O|O(n)]] time, as opposed to $O(n^2)$ for brute force.

---
# Notes:

### Same Direction
- Problems for processing or scanning data in one pass
- E.g., [[Fast and Slow Pointers]]
- [[Sliding Window]]

### Opposite Direction
- Find pairs (e.g., 2sum on sorted array)
- Comparing values against each other from the other array.
	- E.g., [[Leetcode 42 Trapping Rain Water]]
