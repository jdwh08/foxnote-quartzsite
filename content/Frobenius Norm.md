# Definition:
A [[Norm]] for [[Matrix]] built around the element-wise squared sums which are then square-rooted, aka square root of the [[Matrix Trace]].


..||A||_F = \sqrt{\sum_{i,j} A_{ij}^2} = \sqrt{Tr(AA^T)}$



Used to get the size of a [[Matrix]].

---
# Notes:
Analogous to the [[L2 Norm]] for vectors.
