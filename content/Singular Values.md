# Definition:
The [[L1 Norm|Absolute Value Norrm]] of the [[Eigenvectors and Eigenvalues|Eigenvalues]] of a [[Matrix]].

Geometrically, measures how much "stretching" the matrix transformation does.

# Notes:

Given some matrix $A$, consider $A^T A$.
