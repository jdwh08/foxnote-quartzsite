# Definition:
Suppose we have a potential triangle with three arbitrary sides $a, b, c$.
To have a valid triangle, we must have that $c \le a+b$.

We can apply this to [[Vector Space]] using [[Norm]]: $||u+v|| \le ||u|| + ||v||$

A function which meets this property: $f(x+y) \le f(x) + f(y)$ (AKA Subadditive)
A sequence which meets this property: $a_{n+m} \le a_{n} + a_{m}$
