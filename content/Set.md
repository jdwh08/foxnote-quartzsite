### Definition:
Set is a collection of definite, well distinguished objects. 

You can do set operations 
(in, not in, complement for elements not in set, intersection, difference, union).

Notes

### Examples:
$\mathbb{N}, \mathbb{Z}, \mathbb{R}$ for natural/integer/reals.
$\emptyset$ for empty set.
