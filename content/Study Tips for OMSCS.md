### NOTES:
1. Use Piazza / Slack CONSTANTLY. Read all the posts, and participate!
2. Get a study buddy.
3. Get a Mac???
4. Dedicate specific time and schedule for doing the work. 
5. Start working on assignments when they are released. Assess time for project.
6. Make a list of the concepts that you don't really get, ordered by importance, and then work through them until you understand.


### Course Selection:
1. Start with easier courses. Try to get core courses earlier. If doing two courses, pair an easy with a hard.
2. Take classes you're interested in.

### Work:
1. Read EVERYTHING! (Syllabus, Lecture Notes, Piazza Links). Theory and prior work is as important as lectures.
2. Think through the problem solving process at a high level, and THEN code.
3. Get the file names right (copy paste!)
4. Keep track of coding experiments. Keep track of data prep, parameters, code/debug for the report writing later.
5. Believe in yourself, have integrity. Start assignments early. Read assignment instructions VERY carefully.
