# Definition:
The [[Probability]] that some sequence $X_n$ approaches some value $x$ approaches 1 as $n$ grows large.

$$P(\underset{n\rightarrow\infty}{\lim} X_n=x)=1$$

---
# Notes:
I.e., situations in [[Probability Space]]: For all possible outcomes, $X_n$ approaches some X.
$$P\left(\omega \in \Omega: \lim_{n\rightarrow\infty} X_n(\omega) = X(\omega)\right)=1$$

### Properties
1. If something is Almost Sure Convergence, then it is definitely [[Convergence in Probability]] by Fatou's Lemma. This also means [[Convergence in Distribution]].
