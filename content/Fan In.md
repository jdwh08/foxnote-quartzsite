# Definition:
The number of inputs for a neuron in a [[Artificial Neural Network|Neural Network]].

Contrasted with [[Fan Out]].

---
# Notes:
Number of inputs vs outputs gives a proxy for dimension changes.
- E.g., A situation in [[Artificial Neural Network|Neural Network]] layers where the number of outputs is ***smaller*** than the number of inputs, i.e., the dimension "fans in".

Useful for setting activation functions based on architecture
- [[Xavier Weight Initialization]]
- [[Kaiming Weight Initialization]]
