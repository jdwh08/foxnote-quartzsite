# Definition:
Given a list of $n$ items with a size and value, determine the largest value within some size constraint. 

---
# Notes:

This problem is weakly [[NP-Complete]] since there is a [[Dynamic Programming]] solution in $O(nk)$ time, though it takes exponential [[Big O#Runspace Analysis|Space Complexity]].
