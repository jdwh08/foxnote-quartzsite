Eigenvectors and Eigenvalues

Eigenvectors and Eigenvalues

Mar 29, 20245 min read

  • math/linear_algebra

Definition:

Given a Square Matrix A, consider Ax=λx. x are the EIGENVECTORS which maintain their same direction after the transformation. λ are the EIGENVALUES which multiply x.

Geometric Interpretation, 3B1B

Recall that a square matrix A,dim(A)=Rn×n represents a linear transformation of vectors from Rn to Rn.

EIGENVECTORS are the vectors x which maintain their direction after the linear transformation, and EIGENVALUES represent how much the transformation scaled the vector.

Notes:

Strang’s Fun Facts

Square matrixes multiply vectors, and we are interested in vectors that keep their same direction.

Obviously, x=0 because this is always true for the zero vector. Finally, λ∈R.

  • When λ=0, we are finding Ax=0, which is equivalent to the Matrix Null Space.
  • We NEED to have something other than the 0 in the Null Space for this!
  • Thus, if A is a Singular Matrix the transformation allows values into the Null Space.

A Projection Matrix in 3-d have eigenvectors which are in the plane, and the one vector which is perfectly perpendicular.

The sum of the λ values equals the Matrix Trace (the sum of the diagonal from top-left to bottom-right). The determinant is also the product of the eigenvalues.

Transformations:

  • Adding 3I to a matrix changes its eigenvalues by +3, but does not change its eigenvectors: (A+3I)x=λ+3x
  • However, you can’t generalize this: (A+B)x=(λ+α)x doesn’t work because we can’t be sure the eigenvectors for A,B are the same.

You can have imaginary eigenvalues if the matrix gets close to being not symmetric (see example) You can have cases where eigenvalues and thus eigenvectors are repeated (see example).

Examples:

Normal Example Suppose we have some square matrix

A = \begin{bmatrix} 0 & 1 \ -2 & -3 \end{bmatrix}$

which we would like to calculate the eigenvectors and values of.

We need to find Ax=λx=λIn×nx. Subtract, so Ax−λIx=0, which is nice and easy to work with now. This gives us (A−λI)x=0, and x=0, so by the Zero Product Property we know that A−λI has some value that equals zero.

  • In other words, A−λI has something in the Matrix Null Space besides 0
  • In other words, A−λI is a Singular Matrix and non-invertible, hence det=0.

We can thus calculate A−λI:

A - \lambda I= \begin{bmatrix} 0 & 1 \ -2 & -3 \end{bmatrix} - \begin{bmatrix} \lambda & 0 \ 0 & \lambda \end{bmatrix} = \begin{bmatrix} -\lambda & 1 \ -2 & -3-\lambda \end{bmatrix}$

and this has Matrix Determinant of zero, so −λ(−3−λ)−−2∗1=0→λ2+3λ+2=0. which is our “characteristic equation”

  • Notice how the Trace shows up as the middle term (Tr(A)=−3 vs 3λ)
  • Notice how the Matrix Determinant shows up as the last term (det(A)=2 vs +2) →λ=−1,−2 These are our eigenvalues.

Consider λ=−1 as our eigenvalue. Go back to Ax=λx, and solve Ax=0 so

\begin{bmatrix} 0 & 1 \ -2 & -3 \end{bmatrix} \begin{bmatrix} x_1 \ x_2 \end{bmatrix} = -1 \begin{bmatrix} x_1 \ x_2 \end{bmatrix}$

This gives the equations x2​=−x1​ and −2x1​−3x2​=−x2​. Both of these equations end up with x1​=−x2​, hence the associated eigenvector(s) is x1​∗[1−1​] for any x1​∈R.

Imaginary Eigenvalues Suppose we have some 90 degree rotation matrix. NOTE: there are no vectors other than 0 which preserve their direction, so there should be no eigenvectors.

Q = \begin{bmatrix} 0 & -1 \ 1 & 0 \end{bmatrix}$

The Matrix Trace is 0+0=0=λ1​+λ2​, and the Matrix Determinant is 0∗0−(−1∗1)=1=λ1​λ2​. We see that this polynomial shouldn’t be possible…

  • det(Q−λI)=λ2+1=0, so eigenvalues are i,−i. Uh oh, Complex Numbers since this is a complex conjugate pair.
  • Notice how this matrix is the complete opposite of a Symmetric Matrix, i.e., QT=−Q not Q. The closer we get to symmetric matrix, the larger the real part of the eigenvalue.

Repeated Eigenvalues Consider the eigenvectors/values of

R = \begin{bmatrix} 3 & 1 \ 0 & 3 \end{bmatrix}$

which is a Triangular Matrix. In these cases we can “read the eigenvalues” right off the diagonal, since the values which add to 6 and multiply to 9 are literally right there man. Thus, we have a repeated eigenvalue: λ=3,3.

What happens to the eigenvectors? Consider (R−λI)x=0. We see that

R-\lambda I = \begin{bmatrix} 0 & 1 \ 0 & 0\end{bmatrix} \rightarrow \begin{bmatrix} 0 & 1 \ 0 & 0\end{bmatrix} \begin{bmatrix} x_1 \ x_2 \end{bmatrix} = \begin{bmatrix} 0 \ 0\end{bmatrix}$

so, we get our eigenvectors are… both x1​[10​] so there isn’t a second one. Hence, this is a “degenerative matrix” where the eigenvalues don’t get all the information.
