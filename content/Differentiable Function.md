# Definition:
A function which has its [[Derivative]] defined.

For some point $x=z$, this is true if
1. there exists some [[Derivative]] approaching from the left (i.e., as $z^+$), 
2. there exists some [[Derivative]] approaching from the right (i.e., as $z^-$)
3. the two derivatives are the same.
