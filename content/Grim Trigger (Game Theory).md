# Definition:
A strategy for [[Game Theory]] in repeated games, e.g., [[Prisoner's Dilemma Repeated]], where we cooperate until the other player defects, and then defect FOREVER.

> "Dead Man's Switch"
> GRUDGER

![[_Media/Excalidraw/Grim Trigger Game Theory.png|340x296]]

---
# Notes:

Theory is that the **threat of eternal punishment** will compel good behaviour from the other player.

### Issues
This threat is probably *implausible*, i.e., it really doesn't seem like it is worth it for the GRUDGER player to do this!
- Once it is time for the threat to happen, it would be better for GRUDGER to not follow through.
- We need this threat to be [[Subgame Perfect]], i.e., it would make logical sense to do this in all possible histories.
