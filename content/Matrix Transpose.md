# Definition:
An operation which flips the rows and columns of a [[Matrix]] across the main diagonal.

$$(A^T)_{i,j} = A_{j,i}$$

---
# Notes:

### Properties
- Transpose of [[Matrix Product]]: $(AB)^T = B^T A^T$.
