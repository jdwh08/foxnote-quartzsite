### Definition:
The summation notation $\sum$ has the following properties which come from addition:
1. $\sum_{i=1}^{n} a = (a + a + a + ...) = n*a$ if $a$ is a constant.
2. $\sum_{i=1}^{n} ax_i = (ax_1 + ax_2 + ax_3 + ...) = a*\sum_{i=1}^{n} x_i$
3. Same is true for two random variables.
