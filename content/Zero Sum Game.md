# Definition:

A type of Game in [[Game Theory]] where the rewards of one player are equal to the punishments for another player.

---
# Notes:

While we can technically have zero-sum games for more than two players, it's basically saying $a+b+c=0 \rightarrow a+b=c$ which turns this into a general game with $c$ as fudge factor.
