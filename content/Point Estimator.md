# Definition:
Some function of the data, e.g., [[Sample Mean]], [[Standard Deviation]], etc.

---
# Notes:

### Statistical Interpretations
In the [[Frequentist Statistics]] definition, the population has some true fixed $\theta$ which we estimate from sample data as a [[Random Variable]] $\hat{\theta}$.

### Properties
1. [[Bias (estimator)]]: How far away the [[Expected Value]] is from the actual $\theta$.
2. [[Variance (estimator)]]: How much the estimator $\hat{\theta}$ changes based on our data.
	1. Typically use [[Standard Error]] to build [[Confidence Interval]] and [[Hypothesis Test]].
3. [[Consistency (estimator)]]: Whether the asymptotic value $n \rightarrow \infty$ happens.

#### [[Bias-Variance Trade Off]]
If we are using [[Mean Squared Error|MSE]], we have a tradeoff for estimators as bias squared plus variance.
