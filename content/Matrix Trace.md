# Definition:
The sum of the diagonal values of a [[Matrix]].



..TR(A) = \sum_i A_{i,i}$



---
# Notes:

### Properties
1. Consistent with [[Matrix Transpose]] $Tr(A)=Tr(A^T)$ (duh). 
2. If we have a Matrix which can be decomposed into different parts, the Trace is commutative: $A = \Pi A_i \rightarrow Tr(ABC)=Tr(BCA)=Tr(CAB)$
