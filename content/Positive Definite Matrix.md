# Definition:
A matrix such that $x^T A x > 0$.

Contrast with [[Positive Semidefinite Matrix]] where it can also be zero.

---
# Notes:

### Properties
- $x^T A x \ge 0$. We can think of these matrices as ones which only ever "grow".
- A [[Matrix]] whose [[Eigenvectors and Eigenvalues|eigenvalues]] are all strictly positive.
