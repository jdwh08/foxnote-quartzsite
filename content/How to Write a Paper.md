# Notes:

Get your graphs and tables ready beforehand.
Start with methodologies section, then write your results, then your conclusion, before finally writing the abstract.

#### Rules for Writing
1. Present only one contribution per paper. Put it in the title.
2. Make it understandable for naive readers quickly. 
3. Write using context -> content -> conclusion (CCC).
	1. Context: Gap the paper must fill. What's the issue?
	2. Content: Thing that we do to fill.
	3. Conclusion: What does this mean for the field? 
4. Group related sections together. Use similar "parallel" syntax for these.
5. Abstract should have a mini complete story (with CCC).
6. Introduction should set context by explaining gap and why paper matters. CCC this?
7. Results should be statements that slowly build on each other. 
	1. Focus in particular on figures, tables, and legends.
8. Conclusion should explain how gap is filled, limitations, and relevance.
9. Focus paper on **title**, **abstract**, **figures**, and **outline**.
10. Revise and get feedback.
