### Definition:
**Geometric Interpretation, 3B1B**
Recall that a matrix $A, dim(A)=\mathbb{R}^{m \times n}$ represents a [[Linear Transformation]] of vectors from $\mathbb{R}^m$ to $\mathbb{R}^n$. 
The rank is the number of dimensions of the output of this transformation. E.g., if all the resulting vectors land on a line, it is a 1-D result and thus rank 1.

**Strang Interpretation**
The rank is the number of columns/rows in the matrix which are not [[Linear Dependence|Linearly Dependent]] from each other.
The rank is the number of [[Matrix Pivot]]s after doing [[Matrix Elimination]]. 
The rank is the number of dimensions in the [[Matrix Column Space]].
