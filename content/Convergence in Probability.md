# Definition:
As the sample size increases, the probability of a value being larger than some $\epsilon$ approaches zero.



..P(|\hat{\theta_n} - \theta| > \epsilon) \rightarrow 0 \text{ as } n \rightarrow \infty$



Essentially a stronger version of the [[Bias (estimator)]] at the limit.
