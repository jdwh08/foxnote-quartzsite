# Definition:
A type of algorithm which solves things faster by splitting the input into smaller chunks and then solving those smaller, easier chunks.

Notes

# Examples
- [[Quicksort]] and [[Mergesort]]: an element of length 1 is by definition sorted, so we can build our way up from there.
