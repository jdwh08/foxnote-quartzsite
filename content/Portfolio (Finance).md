# Definition:
A collection of assets.

Formally:
- We have assets $i$
- We have some fraction of total value in each asset $w_i$
- $\sum_{i} abs(w_i) = 1.0$
- Portfolio returns are the weighted average of the asset returns $\sum_i w_i * r_i$
