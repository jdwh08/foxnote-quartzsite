### Definition:
Simple: Some property that remains true in a system for all steps.

Formal: Predicate P such that $P(q_{0})$ holds, whenever $P(q)$ is true for state $q$ , and $q \rightarrow r$ then $P(r)$ holds. 

Useful for [[State Machines]] and [[Proof by Invariant]].
