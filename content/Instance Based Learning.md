# Definition:
A type of [[Machine Learning]] which focuses on using stored datapoints instead of modelling some function.

Notably, it uses [[Nonparametric Statistics]]!

---
# Notes:

Comparison against output
- Fast to "train"
- Simple algorithm
- Conservative to existing data ("memorizes")
	- Can overfit if data is noisy

---
# Examples:
[[K-Nearest Neighbours]]: We save a bunch of historical data, and "guess" what the correct output is by looking at nearby neighbours
