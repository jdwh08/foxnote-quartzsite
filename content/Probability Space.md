# Definition:
Defines a [[Random Variable]] from a random process.

- Sample Space $\Omega$ with the set of all possible outcomes.
- Event space $\mathcal{F}$ represents the possible subsets of outcomes which have a  probability ($\mathcal{F} \subseteq \Omega$)
- [[Probability]] function $P$ assigning probability to each event.

# Examples:

### 1D6
You roll one 6-sided die.
- $\Omega = \{ 1, 2, 3, 4, 5, 6\}$
- $\mathcal{F} = \{1\}, \{2\}, \{4, 2, 6\}, ...$ 
- $P: \{1\}=1/6,\ \{4, 2, 6\}=3/6$
