# Definition:

All else equal, one should generally prefer simpler explanations.

---
# Notes:


#### Application to [[Machine Learning]]:
- We should prefer learners and learning algorithms which are simpler, e.g., to avoid [[Overfitting]] via having a too-complex model. (Blumer et al 1987)
- Models which have a expressiveness / representation capacity can result in [[Overfitting]].
