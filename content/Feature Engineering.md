# Definition:
Pre-processing a set of features to **create a new set of features** that **makes it easier to learn** while retaining as much information as possible.

When we remove features, that is the [[Feature Selection]] problem.

---
# Notes:

More formally...


X \sim \mathbb{F}^N \rightarrow \mathbb{F}^M$

 where typically $M < N$ (reduce the [[Curse of Dimensionality]]).
