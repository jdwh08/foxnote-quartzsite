### Definition:
Everything in a function domain which does not have value 0.

Suppose that we have a real valued function whose [[Function Domain]] is an arbitrary set X. Then $supp(x) = {x \in X : f(x) \neq 0}$.

---
### Notes:
**Finite Support** exists when $f(x)=0$ for all but a finite number of points.
