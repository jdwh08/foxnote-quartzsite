# Definition:
A financial theory arguing that existing market prices take into account existing information.
1. Large numbers of investors.
2. New information arrives randomly.
3. Prices adjust quickly.
4. Prices reflect information (in some way). See the variants below.
	1. Public information: Price/Volume
	2. Public (but harder): Fundamentals valuation metrics
	3. Public (but harder still): Exogenous information (e.g., external shocks)
	4. Private information (e.g., company insiders). This is illegal to use.

#### Variants
- **Weak**: Future asset prices ***cannot*** be predicted via historical price and volume data (OHLCV-style).
	- [[Technical Analysis]] should not be effective
	- [[Fundamental Analysis (finance)]] like [[Company Valuation]]-based models can still work.
- **Semi-Strong**: Asset prices adjust immediately to all *publicly available* information, e.g., financial disclosures.
	- [[Fundamental Analysis (finance)]] doesn't work either (e.g., Value Investing)
- **Strong**: Asset prices adjust immediately to reflect *all information*, e.g., insider knowledge.
	- Implies that even insider trading doesn't work.
	- The only portfolio that works is the market portfolio (due to minimizing fees).

# Notes:

- Academic finance generally supports some variation of the EMH (weak).
	- A graph of the 10 year P/E ratio vs 20 year annualized returns shows a strong relationship. 
- In reality, EMH probably affected to some extent by what the asset is: obscure things might be priced less efficiently.
- EMH provides strong theoretical support for [[Passive Investing]].
- EMH assumption that prices contain information is the underpinning of [[Technical Analysis]].
- In order to get efficiency, we need some sort of [[Active Investing]]...
