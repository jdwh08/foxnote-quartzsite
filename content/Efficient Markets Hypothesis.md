# Definition:
A financial theory arguing that existing market prices take into account existing information.
- **Weak**: Future asset prices ***cannot*** be predicted via historical price and volume data (OHLCV-style).
	- Implies that [[Technical Analysis (finance)]] should not be effective, but [[Fundamental Analysis (finance)]] like [[Company Valuation]]-based models can still work.
- **Semi-Strong**: Asset prices adjust immediately to reflect all *publicly available* information, e.g., financial disclosures.
	- Implies that even [[Company Valuation]]-based analysis doesn't work either (e.g., Value Investing)
- **Strong**: Asset prices adjust immediately to reflect *all information*, e.g., insider knowledge.
	- Implies that even insider trading doesn't work.

# Notes:
Academic finance generally supports some variation of the EMH.
In reality, probably affected to some extent by what the asset is: obscure things might be priced less efficiently.
EMH provides strong theoretical support for [[Passive Investing]].
In order to get efficiency, we need some sort of [[Active Investing]]...
