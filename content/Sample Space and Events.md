### Definition:
The **sample space** $S$ of a random experiment is the set of all
possible outcomes.

An **event** is an outcome or a set of outcomes of a random
experiment. That is, an event is a subset of the sample space.

We will denote the set of all events by $\mathcal{A}$.

---
### Notes:
The set of all events $\mathcal{A}$...
- is a set of all subsets of $S$, or $2^S$ if $\mathcal{A}$ is finite.
- is a [σ algebra](Sigma Algebra) of subsets of $S$ if $\mathcal{A}$ is not finite
---
### Examples:
Two fair coin flips have a sample space $S = \{HH, HT, TH, TT\}$. The events are all subsets of S.
